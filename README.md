# Ink & Bone – Tattoo Studio

Responsive landing page for a fictional tattoo studio, built with HTML5 and CSS3.

## Brief Description

Promotional landing page for Ink & Bone, a tattoo studio based in Barranquilla, Colombia. Features artist profiles, a work gallery, pricing table, and contact info. Minimal dark aesthetic with real Unsplash photos.

## Technologies Used

- HTML5 (semantic tags)
- CSS3 (Flexbox, Grid, Custom Properties, Media Queries)
- Vanilla JavaScript (hamburger menu, scroll header)
- Unsplash (image URLs, no download required)

## Project Structure

```
tattoo-landing/
├── index.html
├── css/
```

## How to Run Locally

Open `index.html` in any modern browser. Requires an internet connection to load Unsplash images.
